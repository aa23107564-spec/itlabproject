## 🎯 调整建议

### 如果还是过曝

把窗户灯光再降低：
```javascript
'窗plight': 0.000001,   // 降低10倍
'窗2plight': 0.000001,
```

或者降低曝光度：
```javascript
toneMappingExposure: 0.01  // 从 0.05 降到 0.01
```

### 如果太暗

提高环境光：
```javascript
<ambientLight intensity={0.00005} />  // 提高10倍
```

或提高曝光度：
```javascript
toneMappingExposure: 0.1  // 从 0.05 提高到 0.1
```

### 调整像素大小

```javascript
<Pixelation granularity={3} />   // 更小的像素（更清晰）
<Pixelation granularity={10} />  // 更大的像素（更复古）
```
